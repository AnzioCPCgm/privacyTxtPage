# 知乎创作者智能助手 隐私政策 (Privacy Policy)

**最后更新日期 / Last Updated: 2026-04-02**

感谢您使用“知乎创作者智能助手”(Zhihu AI Agent) 浏览器扩展。我们极其重视您的隐私，本隐私政策旨在透明地向您说明我们如何处理数据。

## 1. 核心承诺 (Core Commitment)
**我们不收集、不存储、不分享您的任何个人身份信息（如姓名、邮箱、密码、浏览历史等）。** 本扩展的所有核心计算和临时数据处理均在您的浏览器本地以及受信任的第三方 API 服务器之间直接进行。

## 2. 数据收集与使用 (Data Collection and Usage)
为了实现“AI 辅助写作”与“智能对话”的核心功能，本扩展在您**主动触发**特定功能时，会进行以下数据处理：
- **页面内容 (Web Content)**：当您使用侧边栏对话或右键菜单的润色、扩写、生成框架等功能时，扩展会且仅会读取您当前在知乎 (zhihu.com) 编辑器中选中的文本，以及当前问题页面的标题/描述。
- **数据流向**：上述提取的文本上下文将通过加密的网络请求（HTTPS）直接发送至 **DeepSeek API** 模型服务器，以生成 AI 回复。我们在此过程中**不会**在任何自有服务器上截留、存储或记录您的对话内容和文章草稿。

## 3. 本地存储 (Local Storage)
扩展会使用 Chrome 浏览器的本地存储功能 (`chrome.storage`) 来保存您的个人设定，以便在您下次启动时自动应用。这些内容包括：
- 扩展的全局开关状态
- 是否开启“屏蔽活动弹窗”的偏好
- 您选择的 AI 回复模型（如 `deepseek-chat` 或 `deepseek-reasoner`）
- 您设置的默认写作语气（如“专业”、“轻松”等）

上述数据**仅保存在您的设备本地**，不会同步或上传到我们的服务器。

## 4. 远程代码与第三方服务 (Third-Party Services)
- 本扩展**不包含任何远程执行代码**（未处于清单文件中声明的代码）。
- 本扩展依赖于大语言模型提供商 (DeepSeek) 来生成内容响应。您的文本数据将在传输加密的保护下送达其 API 接口。

## 5. 权限说明 (Permissions Justification)
- `storage`: 用于在本地存储您的用户配置。
- `contextMenus`: 用于在右键菜单中提供 AI 写作增强快捷操作。
- `activeTab` / `*://*.zhihu.com/*`: 用于获取当前知乎页面的上下文（草稿文字、标题），以及在页面中注入 UI 面板。
- `*://api.deepseek.com/*`: 用于连接 AI 模型服务器获取流式数据。
