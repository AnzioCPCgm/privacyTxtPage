# WebP转存助手 隐私政策

**最后更新日期**：2026年4月8日

感谢您使用“WebP转存助手”扩展程序（以下简称“本扩展”）。开发团队非常重视您的隐私。本说明旨在向您清晰、透明地解释本扩展如何运作及如何保护您的隐私。

郑重承诺：**本扩展是一款100%基于本地浏览器运行的工具，决不会窃取、收集、存储或向任何外部服务器传输您的个人数据及网页浏览内容。**

## 1. 核心运行机制

本扩展的功能是将网页上的WebP等格式图片转换为标准的PNG或JPG格式。
所有的图像读取、重绘及格式转码过程，均完全依赖于您本地浏览器自带的 `OffscreenCanvas` (离屏画布) 技术在本地内存中完成，**没有任何数据会被发送至云端处理**。

## 2. 数据收集与使用

**本扩展不收集任何用户数据。** 
我们不记录您的浏览历史，不收集您的身份信息、密码，不追踪您的点击行为。

## 3. 权限使用说明

为了提供基础的转化与下载服务，本扩展仅申请了最低限度的必要权限：

- **上下文菜单 (`contextMenus`)**  
  **用途**：仅用于在您右键点击网页图片时，在系统菜单中添加“将图片另存为 PNG”和“将图片另存为 JPG”的快捷选项。
- **下载管理 (`downloads`)**  
  **用途**：当本地图片转码完成后，需要调用此权限唤起浏览器的下载功能，将图片正常保存至您的电脑磁盘中。
- **本地存储 (`storage`)**  
  **用途**：仅用于在您的浏览器本地保存您对扩展的偏好设置（例如：是否开启Alt+点击快捷键、默认保存格式、自定义下载路径名的文本）。此配置通过您的 Chrome 账号进行同步。
- **主机权限 (`<all_urls>`)**  
  **用途**：为了破除浏览器画布的跨域安全机制（CORS），以便能够成功对那些跨域的图片进行重绘转码。  
  **承诺**：本权限**只有在您主动发起点击转换操作时**才会去读取那单张图片的二进制文件，绝对不会且并没有代码去静默监控或读取您浏览的其他网页内容。

## 4. 远程代码声明

本扩展严格遵循 Chrome Web Store Manifest V3 安全规范，**不包含亦未调用任何未经审查的远程代码（Remote Code）**。

## 5. 政策的变更

如果在未来因添加新功能而导致隐私政策发生重大变更，我们会在更新扩展程序时同步更新本说明。对于那些绝不收集用户数据的核心承诺，永远不会改变。
